## Overview

This document defines the problem structure, labels, and content for Module 1 readings (Lectures 1 and 2).

---

## Reading Structure

### Check Yourself (embedded throughout)
- 5-6 per reading
- Mixed formats: MC, calculation, short-answer
- Collapsible answers
- Placed after key concepts to reinforce learning
- Reusable for TPS in lecture slides

### Practice Problems (end of reading)
- 12-16 per reading
- Organized by type: Conceptual → Calculation → Synthesis
- Visible difficulty stars (⭐, ⭐⭐, ⭐⭐⭐)
- Hidden HTML comment with full taxonomy: `<!-- TYPE / DEPTH / O→M→I / ⭐⭐ -->`
- Solutions in separate file

### Problem Label Format

```markdown
<!-- Calculation / Application / O→M→I / ⭐⭐ -->
**5.** ⭐⭐ Two identical stars are observed. Star A appears 9× brighter than Star B...
```
